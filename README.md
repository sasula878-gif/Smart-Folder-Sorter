📦 Smart Folder Sorter Ultra
Smart Folder Sorter — это легкая и мощная утилита для Windows, которая автоматически наводит порядок в папке «Загрузки». Больше никаких завалов из файлов — программа сама разложит всё по категориям сразу после скачивания.

✨ Основные возможности / Key Features
🚀 Полная автоматизация: Работает в реальном времени. Скачали файл — он тут же переместился в нужную папку.

📂 Умные категории:

Presentations — для презентаций (.pptx, .ppt, .pps).

Images — фото и картинки.

Documents — отчеты, таблицы, PDF.

Archives, Video, Audio, Installers, Code.

📥 Работа в трее: Приложение работает в фоне и сворачивается в системный трей (возле часов).

⚙️ Автозагрузка: Программа сама запускается при включении компьютера.

🛡️ Безопасность: Ждет завершения загрузки файла перед его перемещением.

🛠 Установка и использование / Installation
Для пользователей (EXE):
Просто скачайте готовый файл из раздела Releases и запустите его.

Для разработчиков (Python):
Клонируйте репозиторий.

Установите зависимости:

Bash
pip install watchdog pystray pillow
Запустите скрипт:

Bash
python smart_sorter.py
🏗 Сборка в один файл / Build to EXE
Если вы хотите собрать свою версию приложения:

Bash
pyinstaller --noconsole --onefile --name="SmartSorter" smart_sorter.py
